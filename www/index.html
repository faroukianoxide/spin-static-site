Impulse

A demonically fast WASM single page website

Impulse is a tiny web page built on Spin and hosted on Fermyon Cloud. With WebAssembly on the server-side, you can build tiny but high-performance microservices that startup fast and run at near-native speed with little memory footprint.

When you make a request for this page, a WASM module is booted to serve you everything you need, and as you are reading this page, the WASM module would by now be terminated, saving resources. Everything happens almost instantaneously. This page is written in HTML with statically served assets (CSS and images files) using Spin static file server. I intend to keep adding features to this page as I experiment with Wasm. You can check out the code here.
